# Movie Recommendation System

This is a simple Python-based movie recommendation system that suggests movies based on user-selected genres.

## Features
- Genre-based movie recommendations
- Simple console interaction
- Easy to extend with new genres

## How it works
User selects a genre (e.g. action, comedy, drama) and the system returns a list of recommended movies.

## Technologies Used
- Python

## Example Genres
- Action
- Comedy
- Drama
- Sci-fi
- Horror
